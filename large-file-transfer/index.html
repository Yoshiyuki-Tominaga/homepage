Large File Transfer

大容量ファイル送信で困りやすい点をまとめて解決

メール添付の容量制限、共有後の取り消し、相手がダウンロードしたか分からない問題を、URL共有と履歴管理で扱いやすくします。

01

最大5GBまで対応

1ファイル最大5GBまで対応し、複数ファイルもまとめて送信できます。

02

期限と回数を設定

公開され続けるリスクを抑えるため、保存期限とダウンロード回数を指定できます。

03

送信後も管理

不要になった共有は取り消しでき、操作履歴やメール通知で状況を確認できます。
