File Transfer URL

ファイル転送URL化とは

ファイル転送URL化とは、ファイルを直接メールに添付せず、アップロードして共有URLを作成する方法です。 相手にはURLを送るだけなので、大容量ファイルや複数ファイルの受け渡しを管理しやすくなります。

容

メール添付の容量制限を避ける

メールでは送りづらい資料、写真、動画、制作物なども、最大5GBまでURLで共有できます。

管

期限・回数・パスワードを設定

共有URLに保存期限、ダウンロード回数、パスワードを設定し、公開範囲をコントロールできます。

消

送信後も取り消せる

不要になった共有や誤って作成したURLは、後から送信取り消しできます。

FAQ

ファイル転送URL化のよくある質問

無料でファイルをURL化して送る前に確認されやすい内容をまとめました。

ファイル転送URL化とは何ですか？

ファイルをアップロードして、相手に共有できるURLを作成することです。メールにファイルを直接添付せず、URLでファイルを受け渡しできます。

無料でファイルをURL化できますか？

はい。Ftensoでは無料でファイル転送URLを作成できます。現在の主な利用条件は、ファイル転送URL作成50件/月、月間アップロード総量50GB、1ファイル最大5GB、最大10日間保存です。

URL化したファイルは後から取り消せますか？

はい。不要になった共有や誤って作成した共有は、後から送信取り消しできます。期限やダウンロード回数も設定できます。

相手は登録なしでダウンロードできますか？

共有URLを受け取った相手は、ブラウザからファイルをダウンロードできます。業務での資料共有や制作物の受け渡しに使えます。

ファイルを無料でURL化して送る。

Ftensoなら、最大5GBのファイル転送URLを作成し、期限・回数・取り消し・履歴まで管理できます。
