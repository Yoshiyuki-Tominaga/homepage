Free File Transfer

無料でも業務に必要な管理機能を使える

小規模な業務利用や個人利用で、ファイル転送・ファイル受信をすぐに試せます。

01

転送URL 50件/月

小規模な業務利用や個人利用に十分な回数で、ファイル転送URLを作成できます。

02

受信URL 10件/月

相手からファイルを送ってもらうための受信URLも、無料で月10件まで作成できます。

03

月間50GBまで対応

1ファイル最大5GB、保存期間最大10日、1URLあたりダウンロード最大10回、受信URL1件あたり受信最大10回まで利用できます。
