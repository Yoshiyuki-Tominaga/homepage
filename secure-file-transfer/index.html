Secure File Transfer

共有範囲をコントロールできるファイル転送

業務利用では、送れることだけでなく、不要になった共有を止められること、あとから確認できることが重要です。

01

期限設定

共有URLの有効期限を設定し、公開され続けるリスクを抑えます。

02

回数制限とパスワード

ダウンロード回数やパスワードで共有の範囲を管理できます。

03

送信取り消し

不要になった共有や誤送信したURLを後から停止できます。
